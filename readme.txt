This plugin utilizes the MenuBaron API to display high-quality food menus on your restaurant's WordPress site.

== Description ==

This plugin is designed for restaurant owners who wish to have a high-quality presense for their restaurant on the internet.  Edit your menus and inclue high-quality food pictures at MenuBaron.com and the data will be synchronized with your WordPress site in real time!

MenuBaron.com is an online tool for marketing your restaurant, managing online menus, and more!

Every time you edit your menus from inside the MenuBaron "Kitchen", your wordpress site will automatically see your updates in real time.

This plugin pulls menu data in to WordPress using the MenuBaron API.  The data transmitted via the API contains menu details only - no code, javascript or tracking cookies.

== Installation ==

1. Upload `mbwp` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Visit the MenuBaron Settings Page in your WordPress dashboard to enter your API key and activate the plugin.

== Frequently Asked Questions ==

= How do I display a full menu on one of my pages or posts? =
Paste this shortcode anywhere you'd like within a post or page: [mb_fullmenu]

= How do I display my daily specials on every page of my site? =
Go to Appearance >> Widgets and drag the "MenuBaron - Category Widget" in to the sidebar area of your choice.
Set a title and select a menu category, such as "Specials".

= Where do I get my API Key? =
Sign in to the <a href="http://kitchen.menubaron.com/" target="_blank">MenuBaron Kitchen</a> and go to the <a href="http://kitchen.menubaron.com/account.php" target="_blank">Account</a> tab.  Once there, your API Key will be displayed in the API tab.

= What kind of data does MenuBaron tranfer through the API =
The MenuBaron API is a one-way data transfer of menu data such as menu categories, dishes, descriptions, and image URLs.  No data is collected from your website visitors and no scripts or code are sourced remotely.
